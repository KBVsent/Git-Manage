# Git-Manage
基于 [HoshinoBotv2](https://github.com/Ice9Coffee/HoshinoBot) 的用于更新通过Git管理的插件

### 使用前置
```bash
pip install -r requirements.txt
```

### 环境需求
```
python 3.8+
```

### 使用说明
```
在 hoshino/config/__bot__.py 中添加本项目文件名
在群里或私信向 bot 发送 [开启 Git-Manage] 启用插件

开启 Git-Manage      ：启用插件
更新全部仓库            ：更新全部通过Git管理的仓库
```
